Common Issues

Possible causes:

  • Website requires authentication or login
  • Website blocks automated requests
  • URL is incorrect or website is down
  • Website uses advanced bot protection

Solutions:

  • Verify the URL is accessible in a browser
  • Check if the website requires login
  • Try again later if the website is temporarily down
  • Contact us if the issue persists

Common issues:

  • Website requires CORS headers for mobile WebView
  • Website uses features not supported in WebView
  • SSL certificate issues
  • Missing development environment setup

Solutions:

  • Ensure your website supports WebView access
  • Test the website in a mobile browser first
  • Check that all resources use HTTPS
  • Follow the platform-specific setup guide

Possible causes:

  • Download interrupted or incomplete
  • File expired (files are deleted after 7 days)
  • Browser download restrictions
  • Antivirus software blocking the download

Solutions:

  • Try downloading again with a stable connection
  • Use a different browser or incognito mode
  • Check your antivirus quarantine folder
  • Generate a new package if the file has expired

Why this happens:

  • Website content is dynamically loaded with JavaScript
  • PWA files are in non-standard locations
  • Website serves different content to our analyzer
  • Conditional loading based on user agent

What to do:

  • Manually verify your PWA setup
  • Check that files are accessible at standard paths
  • Use browser developer tools to inspect your PWA
  • Contact us with specific details about the discrepancy
